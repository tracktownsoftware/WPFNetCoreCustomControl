To create package:
1) Open developer command prompt window for VS2019preview
2) In the developer command prompt window make \WPFNetCoreCustomControl\Pack the current directory
3) PACKAGEVERSION can be modified in packWpfNetCore.bat
4) At command prompt run packAll.bat
5) Nupkg files should be output to \WPFNetCoreCustomControl\Pack\

Note: The WPFNetCoreCustomControl project SDK file contains package information.
